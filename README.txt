To set up your VS Code,

1. Download file
2. Open file in VS code
3. Open terminal
4. Install flask
    "pip install flask"
    "pip install flask-cors"
5. Adjusts the necessary lines in config.py
6. Create database in mariadb
    "create database infmaindb;"
    run sql files (refer to lab 2 for instructions)
7. Run main.py
8. Open "http://127.0.0.1:5000/test_connection" in your browser

It should say "This is working" and done. If you encounter any error in the process, do let me know.

Once it's working, load the file "inventory_purchasing.sql" in MariaDB
